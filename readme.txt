Download Daily comics from ComicsGoneBad.com to your site and embed it to your site

== Description ==

This plugin will download, store and embed the latest daily comics from ComicsGoneBad main site and allow you to add the comics to your post by a simple tag ( [cgb] ) and also allow adding of the latest image on the sidebar as a widget<br />
Please note that only the latest issue of daily comics will be downloaded and shown.<br />
To insert the comics in your post just add the tag [cgb] wherever you want the comics to appear on your blog

== Installation ==


1. Upload `cgb` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make the size settings from the ComicsGoneBad tab in settings of admin panel
4. To add widget to your side bar, goto Appearance> Widget
5. Make sure the comic/ folder within the plugin folder has all write (0777) privileges
